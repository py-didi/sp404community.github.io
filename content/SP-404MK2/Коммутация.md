## Постановка задачи
Представим весь сетап как некоторую цепь, на входе которой источник звука, в качестве звеньев цепи - устройства, звук видоизменяющие и обрабатывающие, а на выходе - колонки или какое-либо устройство для экспорта получившегося материала. 
В зависимости от того, какую конкретную роль в этой цепи будет выполнять Roland SP-404MK2, он может быть закоммутирован различными способами. 

>[!tip]- аудио схема SP-404MK2
>Углублённо разобраться с **внутренним** роутингом сигналов в самом сэмплере может помочь его [[Аудио схема]]. 

Сначала рассмотрим разъёмы Roland SP-404MK2.

## USB Type-C
Данный разъём, при использовании Type-C Data Cable, работает в обе стороны: позволяет принимать звук из внешних источников, а также передавать звук из роланда наружу. В связке с компьютером и DAW (например, Ableton Live) позволяет использовать ваш SP-404MK2 в качестве аудиоинтерфейса. Сигнал по Type-C идёт в две стороны одновременно: роланд может как отправлять звук в телефон / рекордер для записи, так и сам получать из этого же телефона звук для сэмплирования и обработки.
Режим работы Type-C можно выбрать в настройках Utility / System / General / USB-In

MIDI данные тоже передаются по USB Type-C.
#### Задачи, которые можно решить по Type-C
- подключить телефон / плеер / ноутбук / другое совместимое с Type-C устройство в качестве **источника звука** (ютуб, стриминги, приложения). Для записи сигнала извне в роланд.
- подключить телефон / компьютер / рекордер /  прочее оборудование, чтобы записать получившийся в роланде звук куда-либо еще, экспортировать результат своей работы.
- (опционально) при подключении к павербанкам или компьютеру, Type-C также выполняет функцию питания. См. [[Питание]]
#### Требуемые провода
- USB Type-C Data Cable формата "папа-папа" (**именно Data**, они бывают еще и Charge, только для зарядки, которые не передают данные!)

## MIDI In, Out
Порты которые передают MIDI сигнал. Миди вход, позволяет принимать миди-данные извне в Роланд. Миди выход - позволяет роланду передавать сигналы наружу.
#### Задачи, которые можно решить по MIDI
- Управление Роландом с помощью [[MIDI-контроллеры|контроллеров]] / секвенсоров / синтезаторов. Например, подключение клавиатуры для игры в Chromatic mode значительно повысит удобство использования. 
- Управление другим аппаратом через Roland: например назначение knob'ов роланда в качестве крутилок в DAW, или управление посредствам пэдов роланда другим синтезатором в цепи.
- Синхронизация темпа, старта, стопа с другими устройствами по MIDI-clock
#### Требуемые провода
Для подключение потребуются 3,5 Jack TRS Type-A миди кабели с одной стороны, и подходящие для вашего устройства MIDI кабели с другой (например 5-pin MIDI кабель). В отдельных случаях потребуются переходники (переходник с 5-pin DIN кабеля, на джек, или обратно). Перед подключением лучше уточнить, умеет ли ваше устройство получать /передавать миди.

## Line In – линейные входы
В качестве линейных входов в роланде реализована стереопара на джеках. Что позволяет записывать стерео сигнал.

Стерео-сигнал — это звук, который передаётся **двумя независимыми каналами**: левым (L) и правым (R).

>[!info] Простыми словами
>**Моно** — всего один канал, звук одинаковый из всех колонок.
>**Стерео** — два канала, звук может двигаться и разделяться по сторонам. 
>    **стерео = ширина и объём звука**.

Если подключить только канал L, получим моно-сигнал. Сумму левого и правого канала с панорамой по центру. Подключение только R канала приведет к получению звука только в правом ухе, игнорируя левый канал.
Само собой, подключение стереопары целиком дает нам полную объемную картину: левый и правый канал в стерео.
#### Решаемая задача
Запись внешних источников: винилового проигрывателя, магнитофонов, гитар, синтезаторов, получение сигнала из DAW.
#### Требуемые провода
Гитарные провода Jack 6,3 вполне подойдут для работы.

## Line Out – линейные выходы
Аналогичная стереопара на основе Jack разъемов, но для вывода звука наружу, будь то колонки, или постобработка DAW, или другое устройство для дальнейшей работы (например вывести звук из Роланда в Дигитакт)
Аналогично линейным входам, подключив только L канал, мы получим суммированный моно-сигнал. 
Единственная задача - вывод звука из роланда либо в колонки, либо для последующей обработки следующим звеном в цепи (другой семплер, микшер, усилитель, что-угодно еще)
#### Требуемые провода: 
Гитарные провода Jack 6,3 вполне подойдут для работы.

## Гитарный / микрофонный вход
На **передней** панели устройства есть Jack 6,3 вход, в который можно подключать микрофон или гитару. В отличии от стереопары на задней панели устройства, этот вход имеет отдельную ручку усиления (Gain), которая позволяет управлять громкостью входящего сигнала. 

>[!info] Без фантомного питания
При подключении микрофона, модели, требующие **фантомного питания** — а это большинство конденсаторных микрофонов) — не подойдут.

## Разъёмы для наушников
На **передней** панели слева можно найти два выхода на наушники: Jack 6,3 и Jack 3,5 (обычный AUX кабель), которые также позволяют подключить устройства для вывода звука.

## Варианты коммутации устройства

### Вариант 1. Базовое сэмплирование
Вероятно самая популярная схема: из внешнего источника звука, сэмплируем звук в Роланд и выводим в колонки. Для полноценной работы потребуется: 
Вывод: два гитарных кабеля Jack 6,3
Ввод: два гитарных кабеля Jack 6,3 либо Type-C для поддерживающих Type-C устройств. 

Если на входе для нас не важна стерео панорама, можно использовать один кабель, подключенный в L вход. Например, при подключении моно-синтезатора.
![[images/wiring-01.png]]

### Вариант 2. Съёмка видео на телефон
Вывод сигнала и в колонки и в телефон / камеру (для съёмки видео). Набор проводов тот же самый, но телефон выступает не источником звука, а рекордером. 
![[images/wiring-02.png]]

### Вариант 3. Send / Return для обработки внешними эффектами
Позволяет взять звук из Роланда, обработать и вернуть в Роланд. (спасибо [@elonmusic](https://t.me/elonmusic))

Для реализации требуется педаль (или иной обработчик) к правому выходу Line Out
А выход педали подключить к левому входу 
Таким образом, запишется звук с эффектом. Но он будет моно и только левый канал — его нужно будет центрировать.
