## Постановка задачи
Представим весь ваш сетап как некоторую цепь, на входе которой источник звука, в качестве звеньев цепи - устройства, звук видоизменяющие и обрабатывающие, а на выходе - колонки или какое-либо устройство для экспорта получившегося материала. 
В зависимости от того, какую конкретную цель в этой цепи будет выполнять Roland SP-404MK2, он может быть закоммутирован **различными способами**. 

Чтобы лучше понять варианты коммутации рассмотрим возможности Roland SP-404MK2
## Задняя панель устройства

![[sp-404mkii_back.jpg]]

Прежде чем [покупать провода](Провода) и встраивать устройство в цепь, разберемся с тем какие варианты коммутации предлагает само устройство, слева-направо, как на картинке выше: 
### USB Type-C
Данный разьем, при использовании Type-C Data Cable позволяет получать звук из внешних источников, а также передавать звук из роланда наружу.  А также в связке с компьютером и DAW позволяет использовать ваш SP-404MK2 в качестве аудиоинтерфейса. Важно отметить, что работа Type C осуществляется в две стороны одновременно: роланд может как отправлять звук в телефон\рекордер для записи, так и получать из этого же телефона звук в качестве исходника. Выбор режима работы Type C можно выбрать в разделе настроек: utility/system/
#### Задачи, которые можно решить по Type-C:
- подключить телефон/плеер/ноутбук/другое совместимое с Type-C устройство в качестве источника звука (ютуб, стриминги, приложения). Для записи сигнала извне в роланд.
- подключить телефон\компьютер\рекордер\ прочее оборудование, чтобы записать получившийся в роланде звук куда-либо еще, экспортировать результат своей работы.
- (опционально) при подключении к павербанкам или компьютеру, Type-C также выполняет функцию питания. 
### Требуемые провода: 
- USB Type-C Data Cable формата "папа-папа" (**именно Data, они бывают еще и Charge, который передает только питание, а не данные!**)
### MIDI входы:
Как следует из названия, это порты которые передают MIDI сигнал. Миди вход, позволяет передавать миди-сигналы из вне в Роланд. Миди выход - позволяет роланду передавать сигналы во вне. 
### Задачи, которые можно решить по MIDI:
- Управление Роландом с помощью контроллеров\секвенсоров\синтезаторов. Например, подключение клавиатуры, для игры в Chromatic mode значительно повысит удобство использования. 
- Управление другим аппаратом через Roland: например назначение Knob'ов роланда в качестве крутилок в DAW, или управление посредствам пэдов роланда другим синтезатором в цепи.

### Требуемые провода: 
Для подключение потребуются 3,5 Jack миди кабели с одной стороны, и подходящие для вашего устройства MIDI кабели с другой (например 5-pin MIDI кабель). В отдельных случаях потребуются переходники (переходник с 5-pin DIN кабеля, на джек, или обратно). Перед подключением лучше уточнить, умеет ли ваше устройство получать\передавать миди.
## Line-in (Линейные входы)
В качестве линейных входов в роланде реализована стереопара на джеках. Что позволяет записывать стерео сигнал. Для подключения 

> Стерео-сигнал — это звук, который передаётся **двумя независимыми каналами**: левым (L) и правым (R).

> **Простыми словами:**  
>  - **Моно** — один канал, звук одинаковый из всех колонок. - **Стерео** — два канала, звук может двигаться и разделяться по сторонам.
>  - Если кратко: **стерео = ширина и объём звука**.

В случае если подключить только канал L, по умолчанию мы будем получать моно-сигнал. Сумму левого и правого канала с панорамой по центру. Подключение только R канала приведет к получению звука только в правом ухе, игнорируя левый канал. 
Само собой, подключение стереопары целиком дает нам полную объемную картину: левый и правый канал в стерео. 
### Задачи, которые можно решить через линейный вход:
- Запись любых внешних источников: винилового проигрывателя, магнитофонов, гитар, синтезаторов, получение сигнала из DAW.
### Требуемые провода: 
Гитарные провода Jack 6,3 вполне подойдут для работы.
## Line-Out (Линейные выходы)
Аналогичная стереопара на основе Jack разъемов, но для вывода звука наружу, будь то колонки, или постобработка DAW, или другое устройство для дальнейшей работы (например вывести звук из Роланда в Дигитакт)
Аналогично линейным входам, подключив только L канал, мы получим суммированный моно-сигнал. 
Единственная задача - вывод звука из роланда либо в колонки, либо для последующей обработки следующим звеном в цепи (другой семплер, микшер, усилитель, что-угодно еще)
### Требуемые провода: 
Гитарные провода Jack 6,3 вполне подойдут для работы.

## Гитарный\микрофонный вход
На передней панели устройства также есть Jack 6,3 линейный вход, в который можно подключать микрофон или гитару. В отличии от стереопары на задней панели устройства, этот вход имеет отдельную ручку усиления (Gain), которая позволяет управлять громкостью входящего сигнала. 

> Важно отметить, при подключении микрофона, не подойдут модели, требующие наличия **фантомного питания** (Большинство конденсаторных микрофонов).

## Разъемы для наушников
Также на передней панели можно найти два входа под наушники: Jack 6,3 и Jack 3,5 (обычный AUX кабель), которые также позволяют подключить устройства для вывода звука. 

# Варианты коммутации устройства

### Вариант 1. Базовое сэмплирование.
Вероятно самая популярная схема: из внешнего источника звука, сэмплируем звук в Роланд и выводим в колонки. Для полноценной работы потребуется: 
Вывод: два гитарных кабеля Jack 6,3
Ввод: два гитарных кабеля Jack 6,3 либо Type C для поддерживающих Type C устройств. 

Если на входе для нас не важна стерео панорама, можно использовать один кабель, подключенный в L вход. 
![[images/wiring-01.png]]

### Вариант 2: Для съемки видео
Вывод сигнала и в колонки и в телефон\камеру (для съемки видео). Набор проводов тот же самый, но телефон выступает не источником звука, а рекордером. 
![[images/wiring-02.png]]

### Вариант 3: "петля" для обработки сигнала внешними устройствами.
Позволяет взять звук из Роланда, обработать и вернуть в Роланд. (спасибо @elonmusic)

Для реализации требуется педаль (или иной обработчик) к правому выходу line out
А выход педали подключить к левому входу 
Таким образом, запишется звук с эффектом, но он будет моно и только левый канал, его нужно будет центрировать
