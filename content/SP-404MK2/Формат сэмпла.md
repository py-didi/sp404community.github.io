[[index|SP-404MK2]]

Загружать сэмплы в пэды можно двумя способами:
1. импортировать с SD-карточки на самом сэмплере
2. через [[Приложения|приложение]] на компьютере

Внутри SP-404MK2 сэмплы приводятся в формат 16 bit / 48 kHz

### Поддерживаемые форматы

Roland [так пишут](https://support.roland.com/hc/en-us/articles/4408190553883-SP-404MK2-Audio-file-formats-that-can-be-imported) про поддерживаемые форматы сэмплов:

| Импорт с SD-карточки | 16-bit linear, WAV / AIFF / MP3           |
| -------------------- | ----------------------------------------- |
| Через SP-404MK2 App  | WAV / AIFF / MP3 / FLAC / M4A (ALAC, AAC) |

### Частота дискретизации

| Формат                     | bit     | kHz                                                   |
| -------------------------- | ------- | ----------------------------------------------------- |
| WAV / AIFF / FLAC  (*[^1]) | 16 / 24 | 16 / 22.05 / 32 / 44.1 / 48 / 88.2 / 98 / 176.4 / 192 |
| ALAC  (*[^2])              | 16 / 24 | 16 / 22.05 /32 /44.1 / 48 / 88.2 / 98                 |

### Битрейт для сжатых форматов

| Формат   | kbps                                        |
| -------- | ------------------------------------------- |
| AAC [^2] | 128 / 160 / 192 / 224 / 256 / 320           |
| mp3      | 64 / 96 / 128 / 160 / 192 / 224 / 256 / 320 |

[^1]: Через приложение SP-404MK2 App  
[^2]: Только на MacOS
